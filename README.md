# Tourism Management System

Tourism Management System is a web-based application developed as a college mini project.
It helps users explore tourism packages, make bookings, and allows admins to manage
packages and customer details efficiently.

## Features
- User registration and login
- View tourism packages
- Package booking system
- Admin dashboard
- Manage tour packages and bookings
- Customer details management

## Technologies Used
- HTML
- CSS
- JavaScript
- Flask (Python)
- MySQL

## Project Type
College Mini Project

## How to Run (Local)
1. Install Python and Flask
2. Configure MySQL database
3. Run the Flask application
4. Open browser and access the application

## Note
This project is developed for academic purposes and runs locally using Flask.
